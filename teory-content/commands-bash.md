# Basic Commands

**pwd** - Prints the current working directory.

**cd** - Allows you to change the directories.

**/** - You can specify an absolute path.

**..** - Move up to the parent directory.

**ls** - Lists th contents of you current working directory.

**mkdir** - Creates a new directory or folder.

**touch** - Creates a new file.

**mv** - Allows you to move (or rename) a file.

**rm** - Allows you to remove (or delete) a file.

**cp** - Used to copy a file or directory to a new location.

**echo** - Like print() command to print a string to the terminal, but you can actually chain echo to a control symbol.
